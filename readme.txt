CMSMadeSimple Portfolio module v1.0 RC1
---------------------------------------

Written by Alexander Bierbrauer, abierbrauer  polyvision.org

This module generates a page similar to this example page: http://demo.tutorialzine.com/2010/03/sponsor-wall-flip-jquery-css/demo.php

Installation:
-------------

1) Copy the portfolio directory in the CMSMadeSimple module directory
2) copy the CONTENTS of the extra directory to CMS_PATH/uploads/portfolio

Notes: 

* Be sure that in the directory CMS_PATH/uploads/portfolio are the javascript files !
* Be sure that the directory CMS_PATH/uploads/images/portfolio is writeable (should be by default)

Stylesheets:
------------

Have a look at the example.css for a basic style
